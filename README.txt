Aby utworzyć stronę z obrazkami należy uruchomić aplikację main.

Aby przesłać kod na GitHub należy utworzyć repozytorium na stronie GitHub oraz
z terminalu będąc w folderze z projektem należy wykonać poniższe komendy:

'git init'
'git add .' 
'git commit -m "commit"' 
'git branch -M main'
'git remote add origin (link do repozytorium) przykład: https://github.com/nazwa_użytkownika/nazwa_repozytorium.git' ze strony GitHub
'git push -u origin main'
